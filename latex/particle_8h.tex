\doxysubsection{Detailed Description}
Defines the \mbox{\hyperlink{structSimulationParameters}{Simulation\+Parameters}} struct for the simulation\textquotesingle{}s parameters. 

Defines the \mbox{\hyperlink{classParticle}{Particle}} class for molecular dynamics simulation.

The file declares the \mbox{\hyperlink{structSimulationParameters}{Simulation\+Parameters}} struct and the function parse\+Command\+Line to extract and validate user inputs using Boost.\+Program\+Options.

The \mbox{\hyperlink{classParticle}{Particle}} class represents an individual particle in the simulation. It includes properties such as position, velocity, force, and mass, as well as methods for updating its state.
